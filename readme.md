# Personal Portfolio Website

## Live Demo
[View Website] https://personalwebsite-flax-rho.vercel.app

## Overview
This is a personal portfolio website designed to showcase my projects, skills, and background as a Computer Science student. It provides a clean and interactive way for recruiters and peers to explore my work.


## Highlights
- Clean multipage layout
- Home, About, Projects, and Contact sections
- Interactive project cards with pop-up details
- Responsive design for different screen sizes
- Deployed live with **Vercel**

## Technologies Used
- **Next.js**
- **React**
- **Tailwind CSS**
- **Vercel**

## Website Sections

### Home
A short introduction and landing page for visitors.

### About
A brief overview of who I am, my background, and my interests.

### Projects
A showcase of my work with interactive cards and project details.

### Contact
A simple section for visitors to reach out or connect with me.

### Purpose
This project was built to create a strong online presence and present my work in a structured, professional format.
